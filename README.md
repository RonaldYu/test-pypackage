# databricks-dataeng

Databricks data engineering Python package.

## Installation

From source (development):

```bash
pip install -e ".[dev]"
```

Build a wheel:

```bash
pip install build
python -m build
```

Wheels and source distribution will be in `dist/`.

## Development

- Source code: `src/databricks_dataeng/`
- Tests: `tests/`
- Run tests: `pytest`
